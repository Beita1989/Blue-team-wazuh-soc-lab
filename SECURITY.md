# Security Policy (Portfolio)

## Scope
This repository is a portfolio lab project. It contains:
- Documentation and reproducible steps
- Sanitized log samples
- Configuration examples

It does **not** contain production secrets or live infrastructure details.

## Reporting
If you find a security issue in this repository content (e.g., accidental sensitive data):
- Open an Issue with the label **security**
- Do not post real credentials, tokens, or personal data

## Sensitive data
All IPs/hosts/usernames shown in logs/screenshots are sanitized or illustrative.
If you believe something is not sanitized, report it and it will be removed.

## Supported versions
Only the latest commit on `main` is maintained for this portfolio.
