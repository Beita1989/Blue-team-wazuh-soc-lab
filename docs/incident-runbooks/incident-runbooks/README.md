# Incident Runbooks

This folder contains SOC-style incident response runbooks for the lab (Wazuh + Sysmon).

## Runbooks

- [IR-001 Suspicious PowerShell](./IR-001-suspicious-powershell.md)

## Conventions

- Each runbook includes: triage → scoping → containment → eradication → recovery → lessons learned.
- Evidence to attach (when available): Wazuh alert ID(s), Sysmon Event IDs, command line, user, host, hash, and timeline notes.
